# 数学分析知识库

## 第一章 函数与极限

### 1.1 函数的基本概念
- **定义**：函数是两个集合之间的一种对应关系
- **性质**：单调性、奇偶性、周期性、有界性

### 1.2 极限的定义与性质
- **ε-δ定义**：∀ε>0, ∃δ>0, 当0<|x-a|<δ时，|f(x)-L|<ε
- **极限运算法则**：
  - 和差法则：lim(f±g) = lim f ± lim g
  - 乘积法则：lim(f·g) = lim f · lim g
  - 商法则：lim(f/g) = lim f / lim g (lim g ≠ 0)

## 第二章 导数与微分

### 2.1 导数的定义
f'(x) = lim_{h→0} [f(x+h)-f(x)]/h

### 2.2 基本求导公式
- (xⁿ)' = n·xⁿ⁻¹
- (sin x)' = cos x
- (cos x)' = -sin x
- (eˣ)' = eˣ
- (ln x)' = 1/x

### 2.3 求导法则
- 和差法则：(u±v)' = u'±v'
- 乘积法则：(uv)' = u'v + uv'
- 商法则：(u/v)' = (u'v - uv')/v²
- 链式法则：(f(g(x)))' = f'(g(x))·g'(x)

## 第三章 积分学

### 3.1 不定积分
∫f(x)dx = F(x) + C，其中F'(x) = f(x)

### 3.2 基本积分公式
- ∫xⁿ dx = xⁿ⁺¹/(n+1) + C (n≠-1)
- ∫eˣ dx = eˣ + C
- ∫sin x dx = -cos x + C
- ∫cos x dx = sin x + C
- ∫1/x dx = ln|x| + C
